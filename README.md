# Pipeline 数据库可视化工具

这是一个用于查看和分析 Pipeline 软件生成的 SQLite 数据库的 Electron 应用程序。

## 功能特性

- 📊 **数据库浏览**：可视化查看所有数据表和数据
- 🔍 **搜索功能**：快速搜索表和内容
- 📋 **表结构查看**：查看每个表的详细结构信息
- 🔗 **关联数据**：自动显示关联表的数据（如 frames 和 ocr_text）
- 📄 **分页浏览**：支持大量数据的分页查看
- 🎯 **排序功能**：点击列标题进行排序
- 📝 **详情面板**：查看单条记录的详细信息

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **Vue 3** - 前端框架
- **Vite** - 构建工具
- **better-sqlite3** - SQLite 数据库驱动

## 安装

```bash
npm install
```

## 开发

```bash
npm run electron:dev
```

这将启动 Vite 开发服务器和 Electron 应用。

## 构建

```bash
npm run electron:build
```

## 使用说明

1. 启动应用后，点击"选择数据库"按钮
2. 选择 Pipeline 数据库文件（通常位于 `C:\Users\<用户名>\.screenpipe\db.sqlite`）
3. 从左侧表列表中选择要查看的表
4. 在中间区域查看数据，可以：
   - 点击列标题进行排序
   - 使用分页控件浏览数据
   - 点击"详情"按钮查看单条记录的详细信息
   - 点击"显示表结构"查看表的列信息

## 数据库结构

该工具支持查看 Pipeline 数据库中的所有表，主要包括：

- **frames** - 屏幕截图帧数据
- **video_chunks** - 视频块文件信息
- **ocr_text** - OCR 识别的文本内容
- **audio_chunks** - 音频块数据
- **audio_transcriptions** - 音频转录内容
- **ui_monitoring** - UI 监控数据
- 以及其他相关表

## 注意事项

- 数据库以只读模式打开，不会修改原始数据
- 对于大型表，建议使用分页功能以提高性能
- BLOB 类型的数据会显示大小信息，不会加载实际内容
